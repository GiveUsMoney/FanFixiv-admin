# FanFixiv 관리자 서버

> 아직 미완성된 문서입니다.

## 컨벤션

```
# 파일, 변수, 함수
snake_case
# 클래스명
CamelCase
# db
snake_case (테이블 앞에는 무조건 tb_를 붙입니다)
```

## 기술 스택

- Python 3.9
- FastAPI

## 설치

```
pip install -r requirements.txt
```

## 실행

```
run.bat
```
